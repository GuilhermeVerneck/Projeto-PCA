# 👤 Eigenfaces — Reconhecimento Facial com PCA

Implementação de um algoritmo de reconhecimento/processamento de imagens faciais utilizando Análise de Componentes Principais (PCA), desenvolvido na disciplina de Álgebra Linear Computacional na Unifesp.

## 📌 Problema

O projeto explora a aplicação do método de Eigenfaces (Autofaces) para reconhecimento facial, com foco em um sistema de controle de acesso baseado em imagens faciais.

## 🛠️ O que foi feito

- Implementação da PCA aplicada a imagens em Python (`EigenFacePCA.py`), calculando autovetores e autovalores
- Geração de representações visuais e gráficos do processamento (`Figure_1.png` a `Figure_4.png`)
- Consolidação da fundamentação teórica e aplicação prática em relatório acadêmico e apresentação

## ✅ Resultado

Algoritmo funcional de Eigenfaces capaz de reduzir a dimensionalidade de imagens faciais e extrair suas características principais, com achados documentados em relatório e apresentação finais.

## ⚙️ Tecnologias

**Linguagem**: Python
**Métodos**: Álgebra Linear Computacional (PCA)
**Visualização**: geração de gráficos (PNG)
**Documentação**: relatório e apresentação em PDF
